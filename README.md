# CCNA Labs

A collection of hands-on Cisco CCNA labs documenting my networking learning journey through practical scenarios built in Cisco Packet Tracer.

The goal of this repository is to demonstrate my understanding of networking fundamentals by designing, configuring, verifying, troubleshooting, and documenting real-world network topologies.

Rather than simply completing exercises, each lab focuses on explaining the networking concepts involved, the implementation process, the verification steps, and the lessons learned.

---

## Repository Goals

- Practice Cisco CCNA topics through hands-on labs
- Build real network topologies using Cisco Packet Tracer
- Document every lab professionally
- Develop troubleshooting and documentation skills
- Create a public portfolio that demonstrates continuous learning
